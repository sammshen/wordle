# wordle

Terminal-Based Wordle in Haskell using Model-View-Controller

----------------------------------------------------------------------------------------------------------

USAGE:

ghc Wordle.hs
1) ./Wordle (Random answer)
2) ./Wordle num (Choose answer between 0 and 2314, found in words.txt)
3) ./Wordle how-to-play (View rules)
